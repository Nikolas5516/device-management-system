# Device Management System

A full-stack web application for tracking company mobile devices,
their details, locations, and assigned users.

## Tech Stack
- **Backend:** C#, ASP.NET Web API, .NET 10, Entity Framework Core
- **Frontend:** Angular 17+, TypeScript
- **Database:** SQL Server (Docker)
- **Auth:** JWT Bearer Tokens
- **AI:** LLM-powered device description generator

## Prerequisites
- Docker Desktop
- .NET 10 SDK
- Node.js 24 LTS
- Angular CLI
